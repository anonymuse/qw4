# Backlog

Status: compatibility index.

This repository uses `docs/coordination/active-board.md` as the active task board. This directory exists so prompts or agents that expect `docs/backlog/` have a stable place to land instead of failing on a missing path.

Use:

- `docs/coordination/README.md` for multi-agent workflow rules;
- `docs/coordination/active-board.md` for current task state;
- `docs/work-packs/` for dated batches of scoped work.

Do not duplicate task state here unless the project decides to adopt a separate backlog format.
